# CR4 Documentation

Welcome to the CR4 documentation repository.

## Getting Started

### Prerequisites
- Node.js (v20 or later)
- npm or yarn

### Installation

\\\ash
npm install
\\\

### Local Development

To start the Mintlify dev server:

\\\ash
npm run dev
# or
mintlify dev
\\\

The documentation will be available at \http://localhost:3000\.

### Building for Production

\\\ash
npm run build
# or
mintlify build
\\\

## Project Structure

- \docs/\ - Documentation pages (MDX files)
- \public/\ - Static assets (logos, images, favicon)
- \mint.json\ - Mintlify configuration
